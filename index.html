基于非线性意见动力学的路径规划

Path Planning Inspired By Nonlinear Opinion Dynamics

2024年6月 - 2025年6月 June 2024 - June 2025

导师: 梅文俊教授, 北京大学工学院

Supervisor: Prof. Wenjun Mei, College of Engineering, Peking University

项目概述:

基于非线性意见动力学理论，开发新型机器人路径规划算法。

主要贡献:

  • 对Naomi Leonard教授小组的"可调敏感度的非线性意见动力学"进行深入文献综述
  • 提出将意见-注意力机制集成到机器人路径规划算法中的研究方案
  • 独立学习机器人路径规划，与高年级学生讨论研究方向
  • 复现原始论文的代码并验证实验结果
  • 修改动力学方程，将意见-注意力机制集成到RRT算法中
  • 解决复杂导航场景中的死锁问题

成果与收获:

  • 因自学和探索能力获得导师认可
  • 获得都灵理工大学暑期研究实习的推荐

Project Overview:

Developed novel robot path planning algorithms based on nonlinear opinion dynamics theory.

Key Contributions:

  • Conducted in-depth literature review on "Nonlinear Opinion Dynamics with Tunable Sensitivity" from Professor Naomi Leonard's group
  • Proposed research scheme integrating opinion-attention mechanisms into robot path planning algorithms
  • Independently studied robot path planning and discussed research directions with senior students
  • Reproduced the original paper's code and verified experimental results
  • Modified dynamics equations to integrate opinion-attention mechanism into RRT algorithm
  • Addressed deadlock issues in complex navigation scenarios

Achievements:

  • Recognized for self-learning and exploration capabilities by advisor
  • Received recommendation for summer research internship at Politecnico di Torino

  • 展示了强大的数学建模能力
  • 巩固了对理论研究和方法开发的兴趣

基于PiERN架构的CFD与扩散吸附科学计算语言融合系统

CFD and Diffusion-Adsorption Scientific Computing-Language Fusion System Based on PiERN Architecture

2025年1月 - 2025年3月 January 2025 - March 2025

课程: 机器学习基础

Course: Fundamentals of Machine Learning

  • 基于PiERN（物理隔离专家路由网络）架构，协作开发融合高精度科学计算与语言推理的混合模型，主导专家模型验证和数据集构建核心工作
  • 复现并验证PDEBench中1D-CFD扩散吸附任务的官方专家模型（FNO、U-Net、PINN），确保其在PiERN框架内独立高效运行
  • 为每个任务设计超100种多样化语言模板（涵盖基础查询、数值修正、条件假设），构建高质量文本到计算模块的训练数据集
  • 为Token Router训练数据集创建50条表达性语句，实现语言推理与科学计算间的动态路由，支持任务自适应调度
  • 在目标任务中实现语言引导的高精度问题求解和可解释推理，验证了PiERN架构在跨物理场景下的有效性和泛化能力
  • 交付完整代码、数据集和实验报告，在期末答辩中展示成果，验证了模型的协同推理性能和计算精度
  • Collaboratively developed a hybrid model integrating high-precision scientific computing and language reasoning based on the PiERN (Physically-isolated Experts Routing Network) architecture, leading core work in expert model validation and dataset construction
  • Reproduced and verified official expert models (FNO, U-Net, PINN) for 1D-CFD diffusion-adsorption tasks from PDEBench to ensure independent and efficient operation within the PiERN framework
  • Designed over 100 diverse language templates per task (covering basic queries, numerical corrections, conditional hypotheses) to build a high-quality training dataset for the text-to-computation module
  • Created 50 expressive statements for the Token Router training dataset, enabling dynamic routing between language reasoning and scientific computing for task-adaptive scheduling
  • Achieved language-guided high-precision problem-solving and interpretable reasoning in target tasks, verifying PiERN's effectiveness and generalization across cross-physical scenarios
  • Delivered complete code, datasets, and experimental reports; presented results in final defense with demonstrated collaborative reasoning performance and computational accuracy

多智能体协作的高考复习系统

National College Entrance Examination Review System with Multi-Agent Collaboration

2025年3月 - 2025年6月 March 2025 - June 2025

导师: 袁坤教授

Supervisor: Professor Kun Yuan

  • 设计并实现了一种新颖的多智能体系统，用于高考复习，利用三个专门的LLM智能体，具有不同的角色：学习者、教师和助教
  • 使用Tkinter开发了一个全面的GUI界面，支持API配置、科目选择、文档上传和实时对话显示
  • 设计了PDF和DOCX格式的文档解析功能
  • 实现了一个结构化的对话工作流程，学习者智能体根据上传的材料提出问题，教师智能体提供解释，助教智能体生成总结性的学习图表
  • Designed and implemented a novel multi-agent system for college entrance examination review, utilizing three specialized LLM agents with distinct roles: learner, teacher, and teaching assistant
  • Developed a comprehensive GUI interface using Tkinter, supporting API configuration, subject selection, document upload, and real-time dialogue display
  • Designed document parsing functionality for PDF and DOCX formats
  • Implemented a structured dialogue workflow where the learner agent asks questions based on uploaded materials, the teacher agent provides explanations, and the teaching assistant agent generates summary learning charts
